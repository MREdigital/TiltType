Install the Keyboard

To install and enable the keyboard system-wide, do the following:

1 - Launch your Settings App
2 - Select "TiltType"
3 - Select "Keyboards", and activate the switch labeled "Keyboard"

Note: You can also quickly reach TiltType settings by popping back to the main screen of this app and selecting "Tweak / Adjust Keyboard Settings".

Select the Keyboard

Once installed, you should see a small globe button somewhere on or around your standard system keyboard. If you press and hold it you will see a popover list which should now contain "Keyboard – TiltType". Select it, and the keyboard should switch to TiltType.

Deselect the Keyboard

To switch back to the standard system keyboard, locate the globe button around the TiltType keyboard and press and hold to select a different keyboard, just as you had before.

Enable Haptics (Vibration)

Starting in iOS 14 Core Haptics only seems to be usable in Keyboard Extensions (like this one) if "Allow Full Access" is granted, by doing the following:

1 - Launch your Settings App
2 - Select "TiltType"
3 - Select "Keyboards", and activate the switch labeled "Allow Full Access"

Note: Although you will be warned that this allows me, the developer, to be able to transmit anything you type, I absolutely will not do that. The keyboard extension stores nothing and makes no network calls of its own. The only thing enabling this option will do is enable haptic feedback when the keyboard is used system-wide.

Use the Keyboard

Drag the joystick and release (or tap directly) to select keys on the keypads. Tap, double tap, or hold the surrounding buttons to trigger their respective behaviors. The "123" and "ABC" keys will switch between the primary and secondary keyboard layouts. The buttons labeled ⇧ (up arrow) provide shift (on single tap) and caps-lock (on double tap) capabilities. The button labeled ← (left arrow) will delete the previous character, and can be held for repeat deletion. The button labeled ⏎ (returning arrow) will insert a new line, and can be held for repeat insertion. The blank buttons each insert a single space, and can also be held for repeat insertion.

Play the Game

As letters drop from above you just need to strike the key corresponding to the nearest letter before they hit your little ship. Harder modes drop more quickly and allow for fewer errors. That's really all there is to it.
